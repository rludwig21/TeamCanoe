As a user, I would like to be able to click on a log and have it fill the fields with the logs information, so I do not have to re-type it.
Once finished editing the log, I would like to be able to update the log or be given the option to not save my changes.

smaller when editing).

As a user, I would like to be able to clear all the fields that are currently filled, in case I want to start a new log. 

As a user, I would like to be able to limit the number of logs shown via searching by date, client, etc.

As a user, I would like to be able to select a database from a dropdown list that does not change.

As a user, I would like the "Time Required" box to automatically select certain ranges of dates for me based on logical workdays.
